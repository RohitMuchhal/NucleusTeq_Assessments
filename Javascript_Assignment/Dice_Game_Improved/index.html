Game Rules

  • Players take turns rolling a dice.
  • If a player rolls a 1, their turn ends and current score resets to 0.
  • If the dice rolls any other number, it is added to the current score.
  • Players can save their score to add it to their saved total.
  • The first player to reach 100 points wins.
  • The Players can edit their name before starting the game.
